## 🎯 About the Project

This portfolio documents the progress of a Final Year Project focused on **Relay Embedded Fault Classification using Deep Learning Methods**. The project aims to develop and validate deep learning techniques for accurate fault classification in relay protection systems.

### Project Objectives:
1. Research fault diagnosis methods in relay protection systems
2. Develop and train deep learning models for fault classification
3. Validate the models with embedded relay data
4. Compare with traditional fault detection methods
5. Document findings and create a comprehensive technical portfolio
